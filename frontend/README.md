# Brain-Sync Frontend

React + TypeScript + Vite 기반 프론트엔드입니다.

## Setup

```bash
npm install
```

## Environment

루트 `.env` 또는 프론트엔드 실행 환경에 다음 값을 설정합니다.

```env
VITE_API_BASE_URL=http://localhost:8000/api
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Current Flow

`/study` 화면에서 다음 MVP 흐름을 시연합니다.

1. PDF 업로드
2. 핵심 개념 추출
3. 질문 생성
4. 답변 평가
5. 단계별 힌트 요청
6. 자기 설명 평가
7. 세션 리포트 조회
